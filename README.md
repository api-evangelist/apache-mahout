# Apache Mahout (apache-mahout)

## APIs

### Apache Mahout
Mahout provides the Samsara linear algebra DSL in Scala/Java for distributed matrix operations, along with algorithm APIs for classification, clustering, recommendation, and dimensionality reduction on Spark and other backends.
